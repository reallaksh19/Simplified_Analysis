# Reporting Contract

The Reporting module is responsible for exporting engineering calculation results into standardized JSON and printable HTML formats.

## Contract
1. Reports must include a mandatory note: "This calculation is a simplified screening/design-aid calculation. It is not a replacement for formal code-compliant pipe stress analysis."
2. Inputs and results should always include their appropriate units.
3. Errors or warnings generated by the solvers or data import layers must be included in the report output.

## Outputs
- **JSON Export**: For machine readability and re-importing test suites.
- **HTML Export**: For professional document viewing and printing.
