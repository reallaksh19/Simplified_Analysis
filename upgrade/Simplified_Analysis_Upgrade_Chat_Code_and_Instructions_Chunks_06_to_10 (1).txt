exportSketchGraphToPCFX steps:
1. Normalize segments.
2. Build connection index.
3. Run validateSketchTopology().
4. Run buildComponentsFromGraphWithDiagnostics().
5. Convert nodes to PCFX canonical node records.
6. Convert fitting nodes to component records.
7. Convert GraphTranslator components to graphTranslatorComponents.
8. Add lossContract entries for non-qualified component data.

Loss contract condition:
If component.derived.componentData.status is MISSING_COMPONENT_DATA or NOT_QUALIFIED, add:
code COMPONENT_DATA_NOT_QUALIFIED
severity warn
componentId
status

graphTranslatorComponents preserve:
TEE_CLASSIFICATION
MAIN_SEGMENT_A
MAIN_SEGMENT_B
BRANCH_SEGMENT

importPCFXToSketchGraph steps:
1. Warn if schema version mismatch.
2. Rebuild nodes with pos/type/rawAttributes/normalized/derived/meta.
3. Merge component metadata back into node meta and derived.pcfxComponent.
4. Rebuild segments.
5. Run validateSketchTopology().
6. Return nodes, segments, diagnostics, lossContract, validation, rawAttributes, normalized, derived.

validatePCFXRoundtrip checks:
- duplicate segment IDs
- fitting node missing after roundtrip
- fitting node type changed after roundtrip
- self-loop segments

Roundtrip error codes:

FITTING_NODE_MISSING_AFTER_ROUNDTRIP
FITTING_TYPE_CHANGED_AFTER_ROUNDTRIP
SELF_LOOP_SEGMENTS_AFTER_ROUNDTRIP
